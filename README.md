# Sila Koyakan Poster Reveal

Laman statik satu skrin dengan tirai interaktif. Tarik/koyak tirai di depan untuk reveal poster di belakang.

Poster asal sudah disalin ke `assets/poster.jpeg`.

Selepas poster reveal, website akan tunggu 3 saat dan redirect automatik ke:

`https://gigventure.onpay.my/order/form/may0326`

## Tukar Poster

Letakkan poster sebenar dalam folder `assets` dengan salah satu nama berikut:

- `poster.png`
- `poster.jpg`
- `poster.jpeg`
- `poster.webp`

Website akan guna fail pertama yang berjaya dimuatkan. Jika tiada fail poster asal, ia akan guna `assets/poster-fallback.svg`.

## Buka

Buka terus `index.html` dalam browser. Tiada build step diperlukan.
